  Project name: Logo (file type: png)                                  Version: 0.262
  ===================================================================================
  This  collection is called "Logo" and consists of images relating to various brands
  of software houses or distributors of arcade videogames. The collection also has  a
  historical value, showing that aesthetic changes also affected the various logos of
  companies and how they changed over the years.
  The collection has been designed and updated constantly by AntoPISA.
  To view this file correctly use the "Courier New" font.
